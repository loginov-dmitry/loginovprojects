GetMax

Определяет максимальные элементы

Объявление функции

  procedure TWorkspace.GetMax(SourAr, MaxElems, Indexes: string;
    ForColums: Boolean);

Назначение

Функция возвращает максимальные элементы массива SourAr и индексы этих элементов. Вместо MaxElems или Indexes можно задать пустую строку - в этом случае соответствующий массив формироваться не будет. Если ForColums=True, то выполнится поиск максимальных элементов для каждого столбца, иначе - для каждой строки.

Примеры

with Base do begin
  // Имеется массив А=[2 8 4; 7 3 9]
  SLoad('A=[2 8 4; 7 3 9]');

  // Определим максимальные элементы и их индексы для
  // каждого столбца
  GetMax('A', 'cEl', 'cInd');
  // В результате получим cEl=[7 8 9]; cInd=[2 1 2]

  // Определим максимальные элементы и их индексы для
  // каждой строки
  GetMax('A', 'rEl', 'rInd', False);
  // В результате получим rEl=[8; 9]; rInd=[2; 3]
end;
