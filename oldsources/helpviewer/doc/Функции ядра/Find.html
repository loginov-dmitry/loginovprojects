Find

Поиск массива

Объявление функции

  function TWorkspace.Find(Name: String): Integer;

Назначение

Функция выполняет поиск массива по его имени Name. Возвращает индекс (порядковый номер) массива в случае успеха или '-1', если массив не был найден. Поиск выполняется в списке имен массивов. Для ускорения поиска применяется алгоритм индексации.

Примеры

Поиск массива 'MyArray' в рабочей области Base
  I := Base.Find('MyArray');
