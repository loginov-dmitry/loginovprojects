DelRef

Удаление ссылки на массив

Объявление функции

  procedure TWorkspace.DelRef(Name: String);

Назначение

Процедура удаляет ссылку на массив Name. Удаление ссылки можно осуществлять только если в другой рабочей области есть ссылка на тот же массив, в противном случае массив никогда не будет удален и будет постоянно находится в оперативной памяти.

Примеры

Удаление из временной рабочей области Ws ссылки на массив 'A':
  Ws.DelRef('A');
