DoError

Генерация исключения

Объявление функции

  procedure TWorkspace.DoError(Msg: String);

Назначение

Процедура генерирует исключение с выводом на экран сообщения Msg. Пользуйтесь этим методом, если в процессе работы вашей функции обнаружились ошибочные данные, с которыми бессмысленно продолжать выполнение этой функции. Во время работы метода DoError() происходит уничтожение всех временных рабочих областей, прерывается выполнение всех подпрограмм.

Примеры

Генерация исключения при делении на ноль

  // Создание рабочей области
  WS := TWorkspace.Create('MyWorkspace'); 

  // Генерация исключения
  if A = 0 then WS.DoError('Ошибка деления на ноль'); 

  B := 1/A; // Деление
В результате на экране появится сообщение "MyWorkspace: Ошибка деления на ноль".
