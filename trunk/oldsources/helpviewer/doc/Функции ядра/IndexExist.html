IndexExist

Проверка существования массива с указанным индексом

Объявление функции

  function TWorkspace.IndexExist(Idx: Integer): Boolean;

Назначение

Функция определяет, существует ли массив с индексом Idx. Возвращает 'True', если да и 'False', если нет.

Примеры

Проверка существования массива с индексом 100 в рабочей области Base
  if Base.IndexExist(100) then
ShowMessage('Массив с указанным индексом существует');
