IsSquare

Проверяет, является ли массив квадратной матрицей

Объявление функции

  1. function TWorkspace.IsSquare(const Name: String): Boolean;
  2. function TWorkspace.IsSquare(const Idx: Integer): Boolean;

Назначение

Функция возвращает True, если массив, заданный своим именем или индексом является квадратной матрицей. В противном случае функция вернет False.

Примеры

// Имеются три массива A=[11]; B=[1 2 3]; 
// C=[1 2 3; 4 5 6; 7 8 9]

// Функция Base.IsSquare('A') вернет True
// Функция Base.IsSquare('B') вернет False
// Функция Base.IsSquare('C') вернет True
