CopyCutCols

Копирование/вырезание строк

Объявление функции

  function TWorkspace.CopyCutCols(SourAr, DestAr: String; ColNumber, Cols: Integer; Cut: Boolean): String;

Назначение

Данная функция может из входного массива SourAr скопировать или вырезать несколько столбцов в выходной массив DestAr. Если Cut=False, то выполняется копирование, иначе - вырезание. Если после вырезания в массиве SourAr не остается строк, то он будет удален из рабочей области. Переменная ColNumber указывает, с какго столбца начать копирование/вырезание, а Cols - сколько столбцов нужно скопировать/вырезать. Функция возвращает имя выходного массива DestAr.

Примеры

1. Копирование нескольких столбцов из массива 'А' в массив 'В':
  Base.CopyCutCols('А', 'B', 2, 5);

2. Вырезание нескольких столбцов из массива 'А' в массив 'В':
  Base.CopyCutCols('А', 'B', 2, 5, True);
