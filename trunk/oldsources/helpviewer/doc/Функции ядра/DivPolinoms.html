DivPolinoms

Деление многочленов

Объявление функции

  procedure TWorkspace.DivPolinoms(DelimName, DelitName, ChastnName, OstatName: String; GaluaValue: Integer=0; ClearGalua: Boolean = True);

Назначение

Функция выполняет деление многочлена DelimName на многочлен DelitName. В результате получается частное ChastnName и остаток OstatName. Деление может выполняться в конечном поле Галуа. Для этого нужно указать степень поля Галуа с помощью аргумента GaluaValue. Если деление в поле Галуа осуществляется в цикле, то для ускорения вычислений рекомендуется установить параметр ClearGalua=False.

Примеры

Разделим многочлен
2X^5 - 13X^3 + X^2 + 15X - 5
на многочлен
X^2 - 5
Сформируем многочлен-делимое и многочлен-делитель. Для этого распишем исходные многочлены более подробно:
2X^5 + 0X^4 - 13X^3 + 1X^2 + 15X^1 - 5X^0,
1X^2 + 0X^1 - 5X^0
Для записи многочлена в форме массива будем просто записывать вподряд все коэффициенты, начиная с нулевой степени:
A=[-5 15 1 -13 0 2];
B=[-5 0 1]
Выполним деление многочленов с получением частного С и остатка О:
  Base.DivPolinoms('А', 'B', 'C', 'O');
В результате получим массивы
C=[1 -3 0 2] и O=[0]
