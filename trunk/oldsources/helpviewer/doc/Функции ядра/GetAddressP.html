GetAddressP

Определение адреса массива

Объявление функции

  function TWorkspace.GetAddressP(Name: String): Pointer;

Назначение

Функция возвращает адрес массива Name в формате "Pointer". Pointer - 4-байтный целочисленный тип данных, в котором хранится смещение относительно текущего сегмента данных. При создании массива операционная система сама назначает адрес его начала (т.е. смещение). Функция определяет именно этот адрес.

Примеры

Определение адреса массива:
  Address := Base.GetAddressP('A');
