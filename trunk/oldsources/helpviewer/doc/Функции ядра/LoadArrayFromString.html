LoadArrayFromString

Загрузка массива из текстовой строки

Объявление функции

  function TWorkspace.LoadArrayFromString(Str: String): String;

Назначение

Функция создает массив в соответствии с данными, находящимися в текстовой строке Str. Возвращает имя созданного массива. Для более подробной информации смотрите описание к функции SLoad().

Примеры

Создание массива A=[1 2 3]:
  Base.LoadArrayFromString('A=[1 2 3]');
