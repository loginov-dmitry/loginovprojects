CopyFromWorkspace

Копирование массива из указанной рабочей области

Объявление функции

  function TWorkspace.CopyFromWorkspace(SourWS: TWorkspace; SourAr, DestAr: String): Integer;

Назначение

Функция копирует массив SourAr из рабочей области SourWS в рабочую область подпрограммы. После копирования массиву назначается новое имя DestAr. Если задан код Base.CopyFromWorkspace(Ws, 'A', 'B'), то из рабочей области Ws в рабочую область Base будет скопирован массив 'A' и ему будет присвоено имя 'B'. Функция возвращает индекс массива DestAr. Данная функция используется для передачи входных массивов в подпрограмму. Рекомендуется пользоваться функцией CopyRefFromWorkspace(), которая копирует не весь массив, а только ссылку на него.
