GetMin

Определяет минимальные элементы

Объявление функции

  procedure TWorkspace.GetMin(SourAr, MinElems, Indexes: string;
    ForColums: Boolean);

Назначение

Функция возвращает минимальные элементы массива SourAr и индексы этих элементов. Вместо MinElems или Indexes можно задать пустую строку - в этом случае соответствующий массив формироваться не будет. Если ForColums=True, то выполнится поиск минимальных элементов для каждого столбца, иначе - для каждой строки.

Примеры

with Base do begin
  // Имеется массив А=[2 8 4; 7 3 9]
  SLoad('A=[2 8 4; 7 3 9]');

  // Определим минимальные элементы и их индексы для
  // каждого столбца
  GetMin('A', 'cEl', 'cInd');
  // В результате получим cEl=[2 3 4]; cInd=[1 2 1]

  // Определим минимальные элементы и их индексы для
  // каждой строки
  GetMin('A', 'rEl', 'rInd', False);
  // В результате получим rEl=[2; 3]; rInd=[1; 2]
end;
