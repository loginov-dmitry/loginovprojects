GetRows

Определение числа строк массива

Объявление функции

  1. function TWorkspace.GetRows(Name: String): Integer;
  2. function TWorkspace.GetRows(Idx: Integer): Integer;

Назначение

1. Функция определяет количество строк в массиве Name.

2. Функция определяет количество строк в массиве, заданном индексом Idx.

Примеры

1. Определение числа строк:
  Rows := Base.GetRows(Name);

2. Определение числа строк:
  Rows := Base.GetRows(Index);
