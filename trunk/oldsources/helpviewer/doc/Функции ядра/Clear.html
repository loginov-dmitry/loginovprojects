Main functions

Clear

Очистка рабочей области

Объявление функции

  procedure TWorkspace.Clear;

Назначение

Процедура удаляет все массивы из рабочей области. При очистке рабочей области происходит очистка списков имен массивов, размеров и адресов. Данный метод применяется в основном только для постоянной рабочей области Base.

Примеры

Очистка временной рабочей области
  Base.Clear();
