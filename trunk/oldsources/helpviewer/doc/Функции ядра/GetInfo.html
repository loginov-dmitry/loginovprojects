GetInfo

Определение параметров массива

Объявление функции

  procedure TWorkspace.GetInfo(Name: String; var Rows, Cols, Address, SizeInBytes: Integer);

Назначение

Процедура определяет для массива Name такие параметры, как число строк Rows, число столбцов Cols, адрес начала массива Address и количество занимаемой памяти в байтах SizeInBytes.

Примеры

Определение параметров массива 'A':
  Base.GetInfo('A', Rows, Cols, Address, Size);
