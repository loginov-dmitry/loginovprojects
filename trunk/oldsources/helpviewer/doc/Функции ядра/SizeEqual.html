SizeEqual

Определяет, равны ли размеры заданных массивов

Объявление функции

  function TWorkspace.SizeEqual(Array1, Array2: string): Boolean;
  function TWorkspace.SizeEqual(Idx1, Idx2: Integer): Boolean;

Назначение

Функция возвращает True, если массивы, заданные своим именем или индексом имеют одинаковые размеры (число строк равно числу столбцов). В противном случае функция вернет False.

Примеры

// Пусть имеются массивы:
// A=[777]; B=[0]; C=[1 2 3]

// Base.SizeEqual('A', 'B') вернет True
// Base.SizeEqual('A', 'C') вернет False
