LoadArrayFromAscii

Создание массива из ascii-файла

Объявление функции

  procedure TWorkspace.LoadArrayFromAscii(const FileName: String);

Назначение

Процедура создает массив из ascii-файла FileName. Ascii-файлы создаются функцией SaveArrayToAscii(). При создании из файла имя массива формируется путем удаления расширения. Если имя файла не соответствует правильному имени массива, то возникнет исключение с выводом соответствующего сообщения.

Примеры

Создание массива из ascii-файла:
  Base.LoadArrayFromAscii('D:\Temp\MatrixData\A.asc');

В результате в рабочей области Base будет создан массив 'A'.
