SetRef

Запись ссылки на массив

Объявление функции

  procedure TWorkspace.SetRef(Ref: TArrayRef);

Назначение

Процедура записывает ссылку на массив, извлеченную с помощью функции GetRef(). Обычно ссылка на массив записывается во временную рабочую область. Массив, переданный по ссылке нельзя удалять, можно удалить только ссылку с помощью метода DelRef().

Примеры

Запись во временную рабочую область Ws ссылки на массив, хранящейся в переменной Ref:
  Ws.SetRef(Ref);
