CopyRefFromWorkspace

Копирование ссылки на массив из указанной рабочей области

Объявление функции

  function CopyRefFromWorkspace(SourWS: TWorkspace; SourAr, DestAr: String): Integer;

Назначение

Функция копирует ссылку на массив SourAr из рабочей области SourWS в рабочую область подпрограммы. После копирования ссылки массиву назначается новое имя DestAr. Если задан код Base.CopyRefFromWorkspace(Ws, 'A', 'B'), то из рабочей области Ws в рабочую область Base будет скопирована ссылка на массив 'A' и ему будет присвоено имя 'B'. Функция возвращает индекс массива DestAr. Данная функция реализует основной способ передачи входных массивов в подпрограмму. Переданный по ссылке массив может использоваться только для чтения. Ваша подпрограмма не должна менять данные в этом массиве.
