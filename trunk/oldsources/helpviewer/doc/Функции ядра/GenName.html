GenName

Генерация уникального имени массива

Объявление функции

  function TWorkspace.GenName(Add: String=''): String;

Назначение

Функция генерирует уникальное имя массива. В рабочей области существует счетчик генерации уникальных имен, который каждый раз увеличивает свое значение на единицу. Уникальное имя строится следующим образом: 'I_'+Значение_счетчика+'_TV'. Максимальное значение счетчика: 9х10^18 (это очень большое число, его емкости хватит на долгие годы безотказной работы программы). Если строка Add не пустая, то генерация уникального имени произойдет без использования счетчика следующим образом: 'I_'+Add+'_TV'.

Примеры

Генерация уникального имени массива:
  TempName := Base.GenName(); // Результат: TempName=I_1_TV
  TempName := Base.GenName('Text'); // Результат: TempName=I_Text_TV
