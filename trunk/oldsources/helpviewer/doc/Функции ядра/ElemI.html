ElemI

Доступ к отдельному элементу массива

Объявление функции

  property  ElemI[Idx, Row, Col: DWORD]: Real read eGeti write ePuti;

Назначение

С помощью свойства ElemI вы можете считать или изменить значение любого элемента массива, заданного индексом Idx. Данное свойство для чтения использует функцию eGeti(), а для записи - функцию ePuti(). Например, если в обычном массиве изменить элемент можно с помощью кода ArrayName[Row, Col] := Value, то в Matrix для этого нужно писать: ElemI[Index, Row, Col] := Value.

Примеры

with Base do begin
  // Имеется массив А, заданный своим индексом aIdx
  // Осуществляем запись в массив:
  ElemI[aIdx, 1, 1] := 777;

  // Чтение из массива
  R := ElemI[aIdx, 1, 1];
end;
