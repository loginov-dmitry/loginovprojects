ElemFast

Доступ к отдельному элементу массива

Объявление функции

  property ElemFast[Adr, Row, Col, ColCount: Integer]: Real
    read GetElemFast write SetElemFast;

Назначение

Данное свойство отличается от свойства ElemI[] тем, что вместо индекса массива нужно указывать его адрес, а также в качестве 4-го параметра необходимо указывать число столбцов. С помощью данного свойства осуществляется самый быстрый доступ к элементам массива в системе Matrix.

Примеры

with Base do begin
  // Имеется массив А. Определяем его адрес и число столбцов
  aAdr := GetAddressI(GetRang('A', Rows, Cols));

  // Запись в массив:
  ElemFast[aAdr, 1, 1, Cols] := 777;

  // Чтение из массива
  R := ElemFast[aAdr, 1, 1, Cols];
end;
