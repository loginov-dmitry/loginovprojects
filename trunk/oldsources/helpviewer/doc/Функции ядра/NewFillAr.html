Main functions

NewFillAr

Создаем массив-диапазон

Объявление функции

  1. function TWorkspace.NewFillAr(Name: String; Rows, Cols: Integer; BaseValue: Real; Step: Extended): String;
  2. function TWorkspace.NewFillAr(Name: String; FirstValue, LastValue: Real; PointCount: Integer): String;

Назначение

1. Функция создает массив размером Rows x Cols и заполняет его с помощью метода FillAr(). Возвращает имя массива Name.

2. Функция создает вектор-строку с указанным числом элементов. FirstValue - значение первого элемента, LastValue - значение последнего элемента, PointCount - требуемое число элементов. Фактически данная функция расчитывает шаг и вызывает функцию FillAr(), в которую передает значение первого элемента и значение шага заполнения. Возвращает имя массива Name.

Примеры

1. Создание массива-диапазона, заполненного последовательностью чисел, начиная с 10 с шагом 5:
  Base.NewFillAr('А', 3, 4, 10, 5);
В результате будет создан массив:
A=
[10 15 20 25
30 35 40 45
50 55 60 65]

2. Создание массива-диапазона с числом элементов равным 10, первое значение равно 1, а последнее равно 1000:
  Base.NewFillAr('А', 1, 1000, 10);
В результате будет создан массив:
A=[1 112 223 334 445 556 667 778 889 1000]
