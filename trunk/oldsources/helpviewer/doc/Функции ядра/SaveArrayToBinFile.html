SaveArrayToBinFile

Сохранение массива в двоичном файле

Объявление функции

  procedure TWorkspace.SaveArrayToBinFile(const FileName: String; Name: String);

Назначение

В отличие от функции SaveToBinFile(), записывающей в двоичный файл все массивы рабочей области, процедура SaveArrayToBinFile() записывает в двоичный файл только указанный массив Name. Если файл FileName не существует, то он будет создан. Если в файле уже содержится массив Name, то при равных размерах нового и старого массивов, новый массив запишется поверх старого, а если размеры разные, то новый массив будет записан в конец файла, а имя старого массива будет изменено таким образом, чтобы оно считалось недопустимым.

Примеры

Сохранение массивов в двоичном файле:
  Base.SaveArrayToBinFile('D:\Temp\MatrixData\Arrays.bin', 'A');
  Base.SaveArrayToBinFile('D:\Temp\MatrixData\Arrays.bin', 'B');
