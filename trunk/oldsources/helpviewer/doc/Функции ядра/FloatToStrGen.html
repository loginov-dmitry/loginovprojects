FloatToStrGen

Преобразование вещественного числа в строку

Объявление функции

  function TMatrixCommonFunc.FloatToStrGen(Value: Extended): string;

Назначение

Функция преобразует вещественное число Value в строку переменной длины. Длина числа зависит от величины его значения. Разделителем целых и дробных разрядов является точка, независимо от настроек операционной системы.

Примеры

Преобразование строки в вещественное число:
  S := MatrixFunc.FloatToStrGen(1.7) // Получим S=1.7;
  S := MatrixFunc.FloatToStrGen(-1.7) // Получим S=-1.7;
