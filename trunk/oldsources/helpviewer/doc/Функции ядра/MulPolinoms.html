MulPolinoms

Умножение многочленов

Объявление функции

  procedure TWorkspace.MulPolinoms(Pol1, Pol2, PolResult: String; GaluaValue: Integer=0; ClearGalua: Boolean = True);

Назначение

Функция перемножает многочлены Pol1 и Pol2. В результате получается вектор PolResult. Умножение может выполняться в конечном поле Галуа. Для этого нужно указать степень поля Галуа с помощью аргумента GaluaValue. Если умножение в поле Галуа осуществляется в цикле, то для ускорения вычислений рекомендуется установить параметр ClearGalua=False.

Примеры

Выполним умножение многочленов
2X^5 - 13X^3 + X^2 + 15X - 5
и
X^2 - 5
Распишем исходные многочлены более подробно:
2X^5 + 0X^4 - 13X^3 + 1X^2 + 15X^1 - 5X^0,
1X^2 + 0X^1 - 5X^0
Для записи многочлена в форме массива будем просто записывать вподряд все коэффициенты, начиная с нулевой степени:
A=[-5 15 1 -13 0 2];
B=[-5 0 1]
Выполним умножение многочленов с получением в результате массива С:
  Base.MulPolinoms('А', 'B', 'C');
В результате получим массив
C=[25 -75 -10 80 1 -23 0 2];
Если полученный массив представить в форме многочлена, то получим:
2X^7 - 23X^5 + X^4 + 80X^3 - 10X^2 - 75X^1 + 25
