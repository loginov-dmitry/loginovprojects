Elem

Доступ к отдельному элементу массива

Объявление функции

  property  Elem[Name: String; Row, Col: DWORD]: Real 
    read eGet write ePut;

Назначение

С помощью свойства Elem вы можете считать или изменить значение любого элемента массива Name, находящегося в строке Row и столбце Col. Данное свойство для чтения использует функцию eGet(), а для записи - функцию ePut(). Например, если в обычном массиве изменить элемент можно с помощью кода ArrayName[Row, Col] := Value, то в Matrix для этого нужно писать: Elem['ArrayName', Row, Col] := Value. Доступ к элементу массива с помощью данного свойства не самый быстрый, т.к. приходится выполняеть поиск индекса массива по его имени. Значительно быстрее работает свойство ElemI, в котором вместо имени нужно указывать индекс массива.

Примеры

with Base do begin
  // Имеется массив MyArray
  // Осуществляем запись в массив:
  Elem['MyArray', 1, 1] := 777;

  // Чтение из массива
  R := Elem['MyArray', 1, 1];
end;
