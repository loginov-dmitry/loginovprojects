ArrayExist

Проверка существования массива

Объявление функции

  function TWorkspace.ArrayExist(Name: String): Boolean;

Назначение

Функция проверяет, существует ли указанный массив в рабочей области. Если да, то возвращает 'True', иначе - 'False'.

Примеры

Проверка существования массива 'MyArray' в рабочей области Base
  if not Base.ArrayExist('MyArray') then
ShowMessage('Указанный массив не найден в рабочей области');
