LoadFromBinFile

Загрузка рабочей области из двоичного файла

Объявление функции

  procedure TWorkspace.LoadFromBinFile(const FileName: String);

Назначение

Процедура загружает в рабочую область массивы из двоичного файла FileName. Сохранение рабочей области в двоичный файл осуществляет функция SaveToBinFile(). Если во время загрузки массива в рабочей области будет обнаружен одноименный массив, то он будет удален. Чтение данных из двоичных файлов происходит столь же быстро, как и запись, поэтому при работе с большими объемами данных рекомендуется пользоваться именно этим форматом.

Примеры

Загрузка рабочей области Base из двоичного файла:
  Base.LoadFromBinFile('D:\Temp\MatrixData\Base.bin');
