LoadArrayFromMemory

Загрузка массива из структуры в оперативной памяти

Объявление функции

  procedure TWorkspace.LoadArrayFromMemory(ArrayName: String; ArrayAddress: Pointer; RowCount, ColCount: Integer; BaseTypeName: String);

Назначение

Процедура осуществляет загрузку массива ArrayName из структуры, которая находится в оперативной памяти в текущем сегменте данных по адресу ArrayAddress. Практически данный метод используется для копирования массивов среды Delphi в массивы формата Matrix. Допустим, что в Delphi массив объявлен следующим образом:

  var DelphiArray: array[1..100, 1..1000] of Integer;

Нам нужно создать массив Matrix, элементы которого должны быть копией элементов массива DelphiArray. Для этого нужна всего одна строчка кода:

  Base.LoadArrayFromMemory('MatrixArray', @DelphiArray, 100, 1000, 'Integer');

В результате будет создан массив MatrixArray размером 100 x 1000, элементы которого соответствуют элементам массива DelphiArray.
В качестве RowCount и ColCount нужно указывать соответственно число строк и столбцов массива Delphi. BaseTypeName - это имя типа массива Delphi. Поддерживаются все базовые типы: Real, Double, Integer, Longint, Cardinal, Longword, Dword, Byte, Shortint, Int64, Word, Smallint, Short, Real48, Single, Extended, Comp и Currency. В независимости от типа элементы массива Matrix всегда занимают 8 байт и хранятся в формате Real.

Примеры

Загрузка массива из структуры в оперативной памяти:
  Base.LoadArrayFromMemory('MatrixArray', @DelphiArray, 100, 1000, 'Integer');
