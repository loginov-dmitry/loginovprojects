SendMatlabCommand

Передача команды системе MATLAB

Объявление функции

  procedure TWorkspace.SendMatlabCommand(Str: String);

Назначение

Процедура передает среде MATLAB команду на выполнение. Если от системы Matrix еще не было обращений к MATLAB, то эта среда будет запущена. Команды, передаваемые системе MATLAB могут быть любые. Если команда состоит из нескольких строк, то строки необходимо разделять с помощью символов #13#10.

Примеры

Передадим в MATLAB команду, заключающуюся в том, что необходимо создать массив определенных размеров и заполнить его в цикле последовательностью чисел:
  Base.SendMatlabCommand('A=zeros(10, 10)'); // Создание нулевого массива
  Base.SendMatlabCommand('for I=1:10'#13#10'for J=1:10'#13#10+
'A(I, J)=I*J'#13#10'end'#13#10'end');
