GetRef

Извлечение ссылки на массив

Объявление функции

  function TWorkspace.GetRef(Name: String): TArrayRef;

Назначение

Функция возвращает ссылку на массив, т.е. структуру типа TArrayRef, которая описывается следующим образом: 
type
  TArrayRef = record
    Name: String;
    Address: Integer;
    Rows, Cols: Integer;
  end;
Таким образом в ссылке содержится имя, адрес массива, число строк и столбцов.

Примеры

Извлечение ссылки на массив 'A':
  Ref := Base.GetRef('A');
