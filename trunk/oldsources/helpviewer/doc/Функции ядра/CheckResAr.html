CheckResAr

Проверка имени выходного массива

Объявление функции

  procedure TWorkspace.CheckResAr(var Name: String; DoANS: Boolean=True);

Назначение

Процедура проверяет, верно ли задано имя массива Name (это имя нельзя задавать в виде текстовой константы). Если вместо имени указана пустая строка, то при DoANS=True выполняется присвоение стандартного имени: Name := 'ans'. Если задано недопустимое имя, то генерируется исключение с выводом сообщения об ошибке. Процедура используется для проверки имени выходного массива.

Примеры

Проверка имени выходного массива:
  ArrayName1 := '';
  ArrayName2 := 'Array 2';
  ArrayName3 := 'Array_3';
  Base.CheckResAr(ArrayName1);
  Base.CheckResAr(ArrayName2);
  Base.CheckResAr(ArrayName3);

В первом случае происходит присвоение ArrayName1 := 'ans'; во втором случае генерируется исключение, т.к. в имени массива присутствует недопустимый символ - пробел; в третьем случае ничего не произойдет.
