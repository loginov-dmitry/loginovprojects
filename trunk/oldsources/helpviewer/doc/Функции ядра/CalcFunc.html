Main functions

CalcFunc

Вычисление функции одной переменной

Объявление функции

   1. function TWorkspace.CalcFunc(SourAr, DestAr, Operation: String): String;
   2. function TWorkspace.CalcFunc(SourAr, DestAr: String; Func: TRealFunc): String;
   3. function TWorkspace.CalcFunc(SourAr, DestAr: String; Func: TExtendFunc): String;

Назначение

1. Данный метод вычисляет функцию одной переменной, заданную с помощью аргумента Operation. Поддерживаются наиболее часто используемые математические функции. К ним относятся:
SIN   -       Синус
ASIN  -       Арксинус
COS   -       Косинус
ACOS  -       Арккосинус
TAN   -       Тангенс
COTAN -       Котангенс
SEC   -       Секант
COSEC -       Косекант
SQRT  -       Квадратный корень
SQR   -       Квадрат числа
ABS   -       Модуль числа
LOG10 -       Десятичный логарифм
LOG2  -       Логарифм по основанию 2
ROUND -       Округление до ближайшего целого
TRUNC -       Отбрасывает дробную часть
FIX   -       Отбрасывает дробную часть
INT   -       Отбрасывает дробную часть
FRAC  -       Отбрасывает целую часть

Функция возвращает имя выходного массива DestAr.

2. Данный метод вычисляет математическую функцию с помощью подпрограммы, ссылка на которую передается в последнем аргументе. Например, если в качестве последнего аргумента указать Tan, то для вычислений будет использована функция Tan(), которая находится в модуле Math (Такие функции, как Sin() и Cos() вы не сможете использовать в данном методе, т.к. они являются системными функциями). Вы можете программировать собственные вычислительные функции, например:
function MyFunc(const R: Real): Real;
begin
   Result := Sin(R) / Sqrt(3);
end;
Тогда, если в качестве последнего аргумента вы укажете имя функции MyFunc, то именно она будет использована для вычислений.

3. Функция аналогична предыдущей, однако использует не Real, а Extended. Этот тип введен по той причине, что в модуле Math существуют много функций работающих только с типом Extended.

Примеры

1. Имеется массив A=[1 2 3]. Вычислить функцию sin(), результат поместить в массив C:
  Base.CalcFunc('A', 'C', 'Sin');
2. Имеется массив A=[1 2 3]. Вычислить функцию tan(), результат поместить в массив C:
  Base.CalcFunc('A', 'C', Tan); // Необходимо сперва подключить модуль Math
