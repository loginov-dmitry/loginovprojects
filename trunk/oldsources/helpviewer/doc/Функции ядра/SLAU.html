SLAU

Решение системы линейных алгебраических уравнений

Объявление функции

  procedure TWorkspace.SLAU(MatrixName, VektorName, ResultName: String);

Назначение

Функция выполняет решение системы линейных алгебраических уравнений вида A x X = B по алгоритму Гаусса. А (MatrixName) - квадратная матрица коэффициентов, В (VektorName) - вектор-столбец (массив свободных членов СЛАУ), X (ResultName) - вектор-столбец (массив решений СЛАУ).

Примеры

Решим с помощью данной функции систему уравнений:
3 x X - 2 x Y = 9;
2 x X + 5 x Y = 25
Выделим массив коэффициентов и массив свободных членов:
A=
[3 -2;
2 5]
B=
[9;
25]
Определим корни системы уравнения:
  Base.SLAU('А', 'B', 'X');
В результате получим массив
X=
[5;
3]
Следовательно X=5, Y=3.
