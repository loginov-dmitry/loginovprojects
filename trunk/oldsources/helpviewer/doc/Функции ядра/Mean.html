Main functions

Mean

Определяет средние арифметические значения для строк или столбцов

Объявление функции

  function TWorkspace.Mean(SourAr, DestAr: string;
    ForColumns: Boolean = True): string;

Назначение

Функция определяет средние арифметические значения для строк (при ForColumns=False) или столбцов (при ForColumns=True) матрицы SourAr. В результате формируется вектор DestAr. В зависимости от ForColumns этот вектор может быть строкой или столбцом

Примеры

with Base do begin
  // Имеется массив А=[0 1 2; 3 4 5]
  SLoad('A=[0 1 2; 3 4 5]');

  // Определим среднее арифметическое для каждой строки
  Mean('A', 'ForRows', False);
  // В результате получим ForRows=[1; 4]

  // Определим среднее арифметическое для каждого столбца
  Mean('A', 'ForCols');
  // В результате получим ForCols=[1.5 2.5 3.5]
end;
