RenameArray

Изменение имени массива

Объявление функции

   function TWorkspace.RenameArray(SourAr, DestAr: String): Integer;

Назначение

Функция присваивает массиву SourAr имя DestAr. Если массив с именем DestAr ранее существовал, то он удаляется. Если массив SourAr не существует, то генерируется исключение с сообщением об ошибке. Функция возвращает индекс массива с именем DestAr.

Примеры

Переименование массива:
   I := Base.RenameArray('MyArray', 'CopyArray');
