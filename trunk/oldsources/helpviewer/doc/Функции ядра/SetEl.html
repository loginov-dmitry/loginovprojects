SetEl

Изменение значения элемента массива

Объявление функции

  1. procedure TWorkspace.SetEl(Name: String; Value: Real; Row, Col: Integer; AutoSize: Boolean=False; AutoCreate: Boolean=False);
  2. procedure TWorkspace.SetEl(Idx: Integer; Value: Real; Row, Col: Integer; AutoSize: Boolean);

Назначение

1. Метод изменяет значение элемента массива Name, расположенного в строке Row и столбце Col. Этому элементу присваивается значение Value. Выполняется проверка допустимости значений Row и Col. Если вы хотите записать новое значение в несуществующей элемент массива, то поставьте AutoSize равным 'True'. В этом случае процедура сможет увеличить размеры массива в соответствии со значениями Row и Col. Если вы установите AutoCreate равным 'True', то если массив Name не существовал, он будет создан с размерами Row x Col. Вместо данного метода вы можете использовать более быстрое свойство Elem.

2. Метод аналогичен предыдущему с той лишь разницей, что массив указываеся не именем, а индексом, поэтому работает быстрее. Вместо данного метода вы можете использовать более быстрое свойство ElemI.

Примеры

1. Присвоение элементу массива, заданного именем, нового значения:
  Base.SetEl('MyArray', 777, 5, 5);

2. Присвоение элементу массива, заданного индексом, нового значения:
  Base.SetEl(1, 777, 5, 5);
