LoadArrayFromTextFile

Загрузка массива из текстового файла

Объявление функции

  procedure TWorkspace.LoadArrayFromTextFile(const FileName: String; Name: String);

Назначение

Процедура осуществляет загрузку указанного массива Name из текстового файла FileName. Запись массивов в текстовый файл может выполнять одна из функций: SaveToTextFile() или SaveArrayToTextFile(). Если файл не найден или элементы указанного массива заданы неправильно, то генерируется исключительная ситуация с соответствующим сообщением. Если в файле нет указанного массива, то ошибки не возникает.

Примеры

Загрузка массива из текстового файла:
  Base.LoadArrayFromTextFile('D:\Temp\MatrixData\Arrays.txt', 'A');
