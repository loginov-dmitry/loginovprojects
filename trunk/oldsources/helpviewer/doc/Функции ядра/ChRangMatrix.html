ChRangMatrix

Изменение размеров матрицы

Объявление функции

   function TWorkspace.ChRangMatrix(Name: String; RowCnt, ColCnt: Integer): Integer;

Назначение

Функция изменяет размеры матрицы Name, в результате чего она принимает новые размеры RowCnt х ColCnt. Данная функция сохраняет структуру массива, т.е. перемещает элементы массива так, чтобы у них остались преждние координаты. Например, если изменить размеры массива A=[1 2 3; 4 5 6] до 1 х 5, то получим массив A=[1 2 3 0 0], а если до 5 х 1, то получим A=[1; 4; 0; 0; 0] (символ ';' - разделитель строк массива). Для изменения размеров вектора рекомендуется пользоваться функцией ChRang(), т.к. она использует дополнительную оперативную память, выделяемую для векторов при их создании.
Функция возвращает индекс массива Name.

Примеры

Имеется массив A размерами 2 х 3
А=
[1 2 3;
4 5 6]
Измененим размеры массива до 3 х 2:
  I := Base.ChRangMatrix('А', 3, 2);
Получим массив A=
[1 2;
4 5;
0 0]
