GenerateGaluaTable

Генерация таблиц сложения и умножения в конечном поле Галуа

Объявление функции

  procedure TWorkspace.GenerateGaluaTable(TableName: String; Value: Integer; TabSum: Boolean=True);

Назначение

Процедура выполняет генерацию таблицы сложения или умножения в конечном поле Галуа степени Value. При TabSum=True строится таблица сложения, а при TabSum=False - таблица умножения. TableName - имя массива, содержащего таблицу Галуа.

Примеры

Создадим таблицы сложения и умножения в поле Галуа степени 5:
  Base.GenerateGaluaTable('TabSum', 5); // Таблица сложения
  Base.GenerateGaluaTable('TabMul', 5, False); // Таблица умножения

В результате получим следующие массивы: 
TabSum=
[0 1 2 3 4;
1 2 3 4 0;
2 3 4 0 1;
3 4 0 1 2;
4 0 1 2 3];

TabMul=
[0 0 0 0 0;
0 1 2 3 4;
0 2 3 4 1;
0 3 4 1 2;
0 4 1 2 3];
