LoadArrayFromBinFile

Загрузка массива из двоичного файла

Объявление функции

  procedure TWorkspace.LoadArrayFromTextFile(const FileName: String; Name: String);

Назначение

Процедура осуществляет загрузку указанного массива Name из двоичного файла FileName. Запись массивов в двоичного файл может выполнять одна из функций: SaveToTextFile() или SaveArrayToTextFile(). Если файл не найден или элементы указанного массива заданы неправильно, то генерируется исключительная ситуация с соответствующим сообщением. Если в файле нет указанного массива, то ошибки не возникает.

Примеры

Загрузка массива из двоичного файла:
  Base.LoadArrayFromTextFile('D:\Temp\MatrixData\Arrays.Bin', 'A');
