FloatToStrExp

Преобразование вещественного числа в строку

Объявление функции

  function TMatrixCommonFunc.FloatToStrExp(Value: Extended): string;

Назначение

Функция преобразует вещественное число Value в строку фиксированной длины с экспоненциальной частью. Длина числа: 23 знака для положительных и 24 знака для отрицательных чисел. Разделителем целых и дробных разрядов является точка, независимо от настроек операционной системы.

Примеры

Преобразование строки в вещественное число:
  S := MatrixFunc.FloatToStrExp(1.7) // Получим S=1.7000000000000000E+000;
  S := MatrixFunc.FloatToStrExp(-1.7) // Получим S=-1.7000000000000000E+000;
