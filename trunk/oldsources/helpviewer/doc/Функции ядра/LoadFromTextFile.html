LoadFromTextFile

Загрузка рабочей области из текстового файла

Объявление функции

  procedure TWorkspace.LoadFromTextFile(const FileName: String);

Назначение

Процедура загружает в рабочую область массивы из текстового файла FileName. Сохранение рабочей области в текстовый файл осуществляет функция SaveToTextFile(). Если во время загрузки массива в рабочей области будет обнаружен одноименный массив, то он будет удален. Чтение данных из текстовых файлов может занять значительное время, поэтому не рекомендуется хранить в таких файлах большие массивы.

Примеры

Загрузка рабочей области Base из текстового файла:
  Base.LoadFromTextFile('D:\Temp\MatrixData\Base.txt');
