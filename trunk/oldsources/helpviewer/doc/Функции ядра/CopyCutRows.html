CopyCutRows

Копирование/вырезание строк

Объявление функции

  function TWorkspace.CopyCutRows(SourAr, DestAr: String; RowNumber, Rows: Integer; Cut: Boolean=False): String;

Назначение

Данная функция может из входного массива SourAr скопировать или вырезать несколько строк в выходной массив DestAr. Если Cut=False, то выполняется копирование, иначе - вырезание. Если после вырезания в массиве SourAr не остается строк, то он будет удален из рабочей области. Переменная RowNumber указывает, с какой строки начать копирование/вырезание, а Rows - сколько строк нужно скопировать/вырезать. Функция возвращает имя выходного массива DestAr.

Примеры

1. Копирование нескольких строк из массива 'А' в массив 'В':
  Base.CopyCutRows('А', 'B', 2, 5);

2. Вырезание нескольких строк из массива 'А' в массив 'В':
  Base.CopyCutRows('А', 'B', 2, 5, True);
