RandomAr

Заполнение массива псевдослучайными числами

Объявление функции

  function TWorkspace.RandomAr(const Name: String): String;

Назначение

Функция заполняет массив Name псевдослучайными числами. Заполнение производится построчно. Функция Randomize() в данном методе не вызывается, т.е. если вы хотите, чтобы числа были случайными, вам придется перед вызовом метода RandomAr() вызывать функцию Randomize(). Функция возвращает имя массива Name. Если массив Name не существует, то будет сгенерировано исключение.

Примеры

1. Заполнение массива псевдослучайными числами:
  Base.RandomAr('А');

2. Заполнение массива случайными числами:
  Randomize();
  Base.RandomAr('А');
