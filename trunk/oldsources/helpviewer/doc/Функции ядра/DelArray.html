DelArray

Удаление указанного массива из рабочей области

Объявление функции

  function TWorkspace.DelArray(Name: String): Integer;

Назначение

Функция удаляет массив с именем Name из рабочей области. Возвращает индекс удаленного массива. Дело в том, что при удалении массива его имя из списка имен не удаляется, а помечается с помощью символа '$'. Если бы имя удалялось из списка, то индексация остальных массивов могла нарушиться. Кроме того, функция NewArray() производит поиск удаленного массива, помеченного символом '$' в списке имен и создает новый массив на месте помеченного. В случае удаления массива функцией DelArray() и последующего создания массива с тем же именем функцией NewArray(), имя нового массива записывается на месте удаленного, т.е. индекс нового массива будет таким же, как и для удаленного. В случае попытки удаления несуществующего массива ошибки не возникает (попытка удаления несуществующего массива довольно частая операция в Matrix).

Примеры

Удаление массива MyArray из рабочей области Base
  I := Base.DelArray('MyArray');
