Main functions

Create

Создание новой рабочей области

Объявление функции

  constructor TWorkspace.Create(Name: string;
TempWorkspace: Boolean = True);

Назначение

Данный метод предназначен для создания новой рабочей области. Имя рабочей области Name может содержать любой текст, однако рекомендуется задавать осмысленные имена. Имя используется для вывода сообщений об ошибках. Если TempWorkspace=True, то создается временная рабочая область, иначе - постоянная. Постоянные рабочие области, в отличие от временных, не разрушаются при возникновении исключения. Существует главная рабочая область - Base. Она создается и разрушается автоматически, поэтому программист не должен о ней заботиться.

Примеры

Создание временной рабочей области с именем 'MyWorkspace'
  WS := TWorkspace.Create('MyWorkspace');
