IsSingle

Проверяет, является ли массив единичным (т.е. состоит всего из одного элемента)

Объявление функции

  1. function TWorkspace.IsSingle(const Name: String): Boolean;
  2. function TWorkspace.IsSingle(const Idx: Integer): Boolean;

Назначение

Функция возвращает True, если массив, заданный своим именем или индексом состоит из одного элемента. В противном случае функция вернет False.

Примеры

// Имеются два массива A=[11]; B=[1 2 3]
// Функция Base.IsSingle('A') вернет True, а
// функция Base.IsSingle('B') вернет False
