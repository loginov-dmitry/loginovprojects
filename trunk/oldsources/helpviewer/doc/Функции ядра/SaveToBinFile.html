SaveToBinFile

Сохранение рабочей области в двоичный файл

Объявление функции

  procedure TWorkspace.SaveToBinFile(const FileName: String);

Назначение

Процедура сохраняет все массивы рабочей области в двоичный файл FileName. Структура двоичного файла следующая: сначала идет заголовок файла (30 байт), затем имя массива (32 байта), число строк (Integer, 4 байта), число столбцов (Integer, 4 байта), закодированный тип элементов (Word, 2 байта), элементы массива (следуют сразу друг за другом, каждый элемент занимает от 1 до 8 байт в зависимости от типа элементов), после чего могут следовать записи остальных массивов. Запись массивов в двоичный файл происходит очень быстро, поэтому этот формат удобно использовать для хранения массивов больших размеров.

Примеры

Сохранение рабочей области Base в двоичный файл:
  Base.SaveToBinFile('D:\Temp\MatrixData\Base.bin');
