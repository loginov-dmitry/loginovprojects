GetEl

Чтение значения элемента массива

Объявление функции

   1. function TWorkspace.GetEl(Name: String; Row, Col: Integer): Real;
   2. function TWorkspace.GetEl(Idx, Row, Col: Integer): Real;

Назначение

1. Метод считывает значение элемента массива Name, расположенного в строке Row и столбце Col. Этому элементу присваивается значение Value. Выполняется проверка допустимости значений Row и Col. Вместо данного метода вы можете использовать более быстрое свойство Elem.

2. Метод аналогичен предыдущему с той лишь разницей, что массив указываеся не именем, а индексом, поэтому работает быстрее. Вместо данного метода вы можете использовать более быстрое свойство ElemI.

Примеры

1. Чтение элемента массива, заданного именем:
  R := Base.GetEl('MyArray', 5, 5);

2. Чтение элемента массива, заданного индексом:
  R := Base.GetEl(1, 5, 5);
