LoadArrayFromMatlab

Чтение массива из MATLAB

Объявление функции

  function TWorkspace.LoadArrayFromMatlab(const Name: String): Boolean;

Назначение

Функция осуществляет чтение массива Name из рабочей области Base среды MATLAB. Если загрузка массива прошла успешно, то функция вернет значение True, а если нет, то False.

Примеры

Передача массива в MATLAB:
  Base.LoadArrayFromMatlab('A');
