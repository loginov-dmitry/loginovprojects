SumPolinoms

Сложение многочленов

Объявление функции

  procedure TWorkspace.SumPolinoms(Pol1, Pol2, PolResult: String; GaluaValue: Integer=0; ClearGalua: Boolean = True);

Назначение

Функция выполняет сложение многочленов Pol1 и Pol2. В результате получается вектор PolResult. Сложение может выполняться в конечном поле Галуа. Для этого нужно указать степень поля Галуа с помощью аргумента GaluaValue. Если сложение в поле Галуа осуществляется в цикле, то для ускорения вычислений рекомендуется установить параметр ClearGalua=False.

Примеры

Выполним сложение многочленов
2X^5 - 13X^3 + X^2 + 15X - 5
и
X^2 - 5
Распишем исходные многочлены более подробно:
2X^5 + 0X^4 - 13X^3 + 1X^2 + 15X^1 - 5X^0,
1X^2 + 0X^1 - 5X^0
Для записи многочлена в форме массива будем просто записывать вподряд все коэффициенты, начиная с нулевой степени:
A=[-5 15 1 -13 0 2];
B=[-5 0 1]
Выполним сложение многочленов с получением в результате массива С:
  Base.SumPolinoms('А', 'B', 'C');
В результате получим массив
C=[-10 15 2 -13 0 2];
Если полученный массив представить в форме многочлена, то получим:
2X^5 - 13X^3 + 2X^2 + 15X - 10
