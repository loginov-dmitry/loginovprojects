MinMax

Определяет минимальные и максимальные элементы

Объявление функции

  1. procedure TWorkspace.MinMax(Name: string; var MinVal, MaxVal: Real);
  2. function TWorkspace.MinMax(SourAr, DestAr: string;
    ForRows: Boolean=True): string;

Назначение

1. Процедура возвращает наименьший MinVal и наибольший MaxVal элементы массива Name.

2. Функция определяет наибольшие и наименьшие элементы для каждой строки (при ForRows=True) или для каждого столбца (при ForRows=False) матрицы SourAr. В результате будет сформирован массив DestAr, одно из измерений которого равно двум. Если ForRows=True, то DestAr будет состоять из двух столбцов, причем в первом столбце будут храниться минимальные элементы, а во втором - максимальные. Если ForRows=False, то DestAr будет состоять из двух строк, причем в первой строке будут храниться минимальные элементы, а во второй - максимальные.

Примеры

with Base do begin
  // Имеется массив А=[2 8 4; 7 3 9]
  SLoad('A=[2 8 4; 7 3 9]');

  // Определим min и max для каждой строки
  MinMax('A', 'ForRows');
  // В результате получим ForRows=[2 8; 3 9]

  // Определим min и max для каждого столбца
  MinMax('A', 'ForCols', False);
  // В результате получим ForCols=[2 3 4; 7 8 9]
end;
