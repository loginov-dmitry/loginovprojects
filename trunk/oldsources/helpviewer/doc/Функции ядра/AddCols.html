Main functions

AddCols

Объединение массивов по столбцам

Объявление функции

   function TWorkspace.AddCols(SourAr1, SourAr2, DestAr: String): String;

Назначение

Функция добавляет к массиву SourAr1 столбцы массива SourAr2, а на выходе формирует массив DestAr. Длина столбцов должна быть одинаковой. Разрешается в качестве одного из входных массивов указывать несуществующий выходной массив - эта возможность позволяет организовать формирование выходного массива в цикле по данным различных входных массивов. Возвращает имя выходного массива DestAr.

Примеры

Входные массивы:
A=
[1   0.1
2 0.2
3 0.3
4 0.4
5 0.5]
B=
[5   0.5
4 0.4
3 0.3
2 0.2
1 0.1]

1. Объединение массивов А и В. Результат заносится в массив С:
   Base.AddCols('А', 'В', 'C');
В результате получим массив С=
[1   0.1   5   0.5
2 0.2 4 0.4
3 0.3 3 0.3
4 1.4 2 0.2
5 0.5 1 0.1]

2. Объединение несуществующего массива С с массивом А. Результат заносится в массив С:
   Base.AddCols('С', 'А', 'C');
В результате получим массив С=
[1   0.1
2 0.2
3 0.3
4 0.4
5 0.5]
