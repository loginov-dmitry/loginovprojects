SaveArrayToTextFile

Сохранение массива в текстовом файле

Объявление функции

  procedure TWorkspace.SaveArrayToTextFile(const FileName: String; Name: String);

Назначение

В отличие от функции SaveToTextFile(), записывающей в текстовый файл все массивы рабочей области, процедура SaveArrayToTextFile() записывает в текстовый файл только указанный массив Name. Если файл FileName не существует, то он будет создан, иначе массив будет записан в конце файла. Если в файле уже содержится массив Name, то он будет стерт из файла путем "забеливания" символами пробела, а новый массив будет записан в конец файла.

Примеры

Сохранение массивов в текстовом файле:
  Base.SaveArrayToTextFile('D:\Temp\MatrixData\Arrays.txt', 'A');
  Base.SaveArrayToTextFile('D:\Temp\MatrixData\Arrays.txt', 'B');
