GetElemCount

Возвращает реальное число элементов в данной рабочей области

Объявление функции

  function TWorkspace.GetElemCount: Integer;

Назначение

Функция возвращает реальное число элементов в данной рабочей области. Массивы, переданные в рабочую область по ссылке, не влияют на число элементов. Функция не занимается подсчетом, числа элементов, она просто считывает значение переменной ElemCount, которая формируется в ходе работы программы.
