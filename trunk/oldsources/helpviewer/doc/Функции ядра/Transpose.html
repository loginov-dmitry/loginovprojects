Transpose

Транспонирование матрицы

Объявление функции

  function TWorkspace.Transpose(SourAr, DestAr: String): String;

Назначение

Функция создает массив DestAr, который является результатом транспонирования (замены строк столбцами) матрицы SourAr. Функция возвращает имя массива DestAr, что дает возможность использовать эту функцию как аргумент при вызове другой процедуры. Например в результате выполнения кода Base.Transpose(Base.Transpose('A', 'A'), 'A') получим исходную матрицу.

Примеры

Транспонирование матрицы:
  Base.Transpose('MyArray', 'TranspArray');
