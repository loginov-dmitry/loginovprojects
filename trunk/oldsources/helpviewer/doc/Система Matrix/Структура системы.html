Структура системы

Структура системы Matrix представлена на рисунке
  • Модуль Lists.pas - использовался в старой версии.
  • Ядро Matrix.pas - основа системы. Данный модуль содержит необходимые функции для работы с массивами. Подробное описание функций смотрите в разделе справки "Функции ядра".
  • Модули внешних функций - сборники подпрограмм. Вместе с системой Matrix поставляются следующие модули внешних функций:
    • Модуль Signals.pas - модуль внешних функций, содержащий функции для обработки сигналов.
    • Модуль NNet.pas - модуль внешних функций, содержащий функции для работы с нейронными сетями.
    • Модуль DemoFunc.pas - модуль внешних функций, содержащий демонстрационные примеры.
    • Модуль Examples.pas - модуль внешних функций, содержащий примеры подпрограмм.
    • Другие модули, появившиеся после написания данной страницы.
  • MATLAB - реализована возможность тесного взаимодействия с системой MATLAB с помощью средств Matrix.
  • Запись/чтение из файлов - сохранение массивов и рабочих областей в файлы и чтение их из файлов.
  • Модули приложения - модули разрабатываемого приложения. К ним по мере необходимости можно подключить любой из модулей системы Matrix.
