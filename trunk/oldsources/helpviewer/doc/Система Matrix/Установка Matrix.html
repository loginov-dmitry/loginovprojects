Установка Matrix

Вначале Matrix был задуман как небольшой файл, который можно просто скопировать в папку с разрабатываемым приложением. На данный момент система Matrix включает более 150 файлов, которые весят более 3-х Мбайт (сюда входит также справочная система). Для того, чтобы использовать Matrix в своих приложениях, необходимо, чтобы среда Delphi видела файлы системы Matrix. Для этого можно воспользоваться одним из способов:

  • Самый простой и надежный способ - скопировать необходимые файлы в папку с разрабатываемым приложением. Обязательно нужно скопировать файлы Lists.pas, Matrix.pas. Остальные модули (сборники подпрограмм или внешних функций) нужно копировать при необходимости. При переносе приложения на другой компьютер никаких проблем не возникнет.
  • Скопировать всю систему Matrix в директорию ...\Borland\DelphiN\Projects\Bpl. Данный способ тоже простой и избавляет от необходимости дополнительного копирования файлов. Однако способ имеет два недостатка. Во-первых, о файлах, лежащих в папке Bpl, зачастую забывают, поэтому держите на всякий случай запасную копию Matrix. Во-вторых, при переносе своего приложения на другой компьютер вы должны позаботиться о том, чтобы на нем был установлен Matrix.
  • Более сложный способ - указать среде Deplphi путь поиска Matrix. Для этого нужно выбрать пункт меню Tools->Environment Options и в появившемся окне перейти на вкладку Library. В полях "Library Path" и "Browsing Path" следует добавить путь к Matrix. Для этого необходимо нажать кнопку с троеточием и в появившемся окне добавить новый путь. Данный способ имеет недостаток: при переносе своего приложения на другой компьютер вы должны позаботиться о том, чтобы на нем был установлен Matrix.
