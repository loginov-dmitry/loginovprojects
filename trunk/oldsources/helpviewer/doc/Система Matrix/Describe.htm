MatriX Project представляет
открытый программный проект
систему матричных вычислений
для Delphi



Предлагаемый продукт позволяет производить матричные вычисления над 2-мерными массивами любого размера.

Программные требования:

- Среда разработки: Borland Delphi 5, 6, 7;
- Операционные системы: Windows 95/98/Me/NT/2000/XP;

Аппаратные требования:

- Система работает с любым процессором при любом количестве оперативной памяти

Возможности продукта:

- Интеграция с любым приложением, написанным в среде Delphi;
- Хранение неограниченного количества 2-мерных массивов любых размеров;
- Высокая скорость вычислений;
- Законченное ядро Matrix позволяет программировать алгоритмы любой сложности;
- Организация вычислений бесконечной вложенности;
- Создание изолированных рабочих областей;
- Взаимодействие с системой Matlab
