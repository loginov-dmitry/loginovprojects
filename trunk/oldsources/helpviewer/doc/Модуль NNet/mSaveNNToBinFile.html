NNet

mSaveNNToBinFile

Запись нейронной сети в двоичный файл

Объявление функции

  procedure mSaveNNToBinFile(Ws: TWorkspace; NNName, FileName: String;
     NewFile: Boolean=True);

Назначение

Процедура осуществляет запись нейронной сети NNName, находящейся в рабочей области Ws, в двоичный файл FileName. При NewFile=True файл, имеющий имя FileName, будет удален. В файл будут сохранены веса, смещения и передаточные функции для каждого слоя нейронной сети

Примеры

Запись нейронной сети в двоичный файл:
  mSaveNNToBinFile(Base, 'Net_I', 'D:\Matrix\Net_I.bin');
