mLoadNNFromMatlab

Загрузка нейросети из рабочей области Base среды Matlab

Объявление функции

  procedure mLoadNNFromMatlab(Ws: TWorkspace; NNName: String);

Назначение

Процедура осуществляет загрузку в рабочую область Ws нейронной сети NNName, находящейся в рабочей области Base среды Matlab. Любая нейросеть в Matlab имеет множество параметров и хранится в структуре типа записи. Процедура считывает только важнейшие параметры, которыми являются массивы весов нейронов, массивы смещений нейронов, номера передаточных функций (сейчас нигде не используется) и количество слоев в нейронной сети. В Matrix все эти параметры хранятся в отдельных массивов, имена которых образуются путем прибавления различных окончаний к имени НС. Вот примерный перечень массивов одной нейросети: 
Net_I_Lay1 - веса первого слоя
Net_I_B1   - смещения первого слоя
Net_I_Fnc1 - передаточная функция первого слоя
Net_I_Lay2 - веса второго слоя
Net_I_B2   - смещения второго слоя
Net_I_Fnc2 - передаточная функция второго слоя
Net_I_LCnt - количество слоев в НС

Примеры

Загрузка нейросети из Matlab:
  mLoadNNFromMatlab(Base, 'Net_I');
В результате в рабочей области Base появится нейросеть Net_I.
