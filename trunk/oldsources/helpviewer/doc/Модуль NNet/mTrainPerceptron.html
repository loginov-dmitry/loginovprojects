mTrainPerceptron

Обучение персептрона

Объявление функции

  procedure mTrainPerceptron(Ws: TWorkspace; X, D: string; epoch: Cardinal;
    err: Real; W: string; E: string; T: string;
    ProcProgress: TTrainProgress = nil);

Назначение

Процедура обучает персептрон. Параметры обучения:
  • X - массив обучающих векторов (обучающая выборка). Число строк соответствует числу входов нейрона, число столбцов - количеству обучающих выборок.
  • D - требуемые значения на выходе. Это вектор-строка, число элементов которого равно количеству обучающих выборок.
  • epoch - максимальное количество циклов обучения.
  • err - требуемая погрешность обучения.
  • W - массив весов (результат обучения). Это вектор-столбец, состоящий из 3-х элементов. Первый элемент - смещение нейрона, остальные элементы - веса синапсов.
  • E - ошибка обучения (выходной массив).
  • T - время обучения (выходной массив).
  • ProcProgress - процедура для визуализации хода обучения - не обязательно
Вы можете создать процедуру для визуализации процесса обучения, например: 
procedure Visualization(epoch: Integer; err, ideal: Real);
begin
....................
end;
В эту процедуру поступают: номер текущей эпохи (цикла обучения) epoch, значение функции сходимости err и значение целевой функции ideal.

Процедура mTrainPerceptron() разработана для демонстративных целей и практических применений может не найти.

Примеры

Пусть имеются входные массивы: 
I=[1 0 0 1;
0 1 1 0];

O=[0 1 1 0];
где I - обучающая выборка, O - требуемые значения на выходе. Выполним обучение персептрона: 
  mTrainPerceptron(Base, 'I', 'O', 50, 0.01, 'W', 'E', 'T');
Здесь 50 - число циклов обучения, 0.01 - погрешность обучения, 'W' - вектор весов синапсов, 'E' - ошибка обучения, 'T' - время обучения.
