mSimNetWithMatlab

Обработка системой Matlab входного вектора с помощью нейронной сети, хранящейся в mat-файле

Объявление функции

procedure mSimNetWithMatlab(WS: TWorkspace; NetPath, NetName, InputArray,
  ResultArray: String);

Назначение

Процедура использует возможность взаимодействия с системой Matlab. Происходит вызов среды Matlab, куда загружается mat-файл с нейронной сетью. Имя этого файла (оно же имя нейронной сети) - NetName, путь к файлу - NetPath. InputArray - передаваемый в Matlab анализируемый вектор-столбец. Ws - ссылка на рабочую область, содержащую входные данные. Преимущество использования данной процедуры для расчета нейросетей состоит в том, что Matlab, вызываемый с помощью этой процедуры, может произвести расчет любой нейросети, вне зависимости от ее топологии.

Примеры

Пусть в папке 'D:\Matlab\Work' имеется файл с нейросетью Net_I.mat. В рабочей области Base системы Matrix имеется анализируемый массив A, представляющий собой вектор-столбец. В данном случае, вызов процедуры будет выглядеть следующим образом:
  mSimNetWithMatlab(Base, 'D:\Matlab\Work', 'Net_I', 'A',  'Out');
В результате в рабочей области Base появится новый массив Out.
