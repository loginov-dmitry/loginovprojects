NNet

mCalcKohon

Расчет нейронной сети LVQ (гибридная сеть Кохонена)

Объявление функции

  procedure mCalcKohon(Ws: TWorkspace; NNName, TestVek, ResVek: string;
    var Per: Real; Euklid: string = '');

Назначение

Процедура расчитывает гибридную сеть с конкурентным слоем Кохонена и сигмаидальным слоем с линейной функцией активации. Указывается имя сети NNName, имя анализируемого вектора TestVek и имя выходного вектора ResVek. Процедура возвращает также степень уверенности распознавания в процентах (аргумент Per) и, при желании, вектор близости (евклидовых расстояний между весами и входами НС) Euklid. Данная нейронная сеть (LVQ) используется для анализа сигналов ЭКГ в 12-ти стандартных отведениях.

Примеры

Анализ входного вектора P нейронной сети LVQ NNet_I:
  mCalcKohon(Base, 'Net_I', 'P', 'Z', Per, 'DistVek');
В результате получим выходной вектор Z, процент уверенности распознавания Per, вектор близости DistVek.
