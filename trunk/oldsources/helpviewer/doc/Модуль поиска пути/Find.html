TFindPath

Find

Выполнение поиска

Объявление функции

  function TFindPath.Find(bRow, bCol, eRow, eCol: Integer; 
    Proc: TFindPathProc=nil): Boolean;

Назначение

Функция выполняет поиск кратчайшего пути между начальной точкой (bRow, bCol) и конечной точкой (eRow, eCol). Кратчайший путь - это минимальный путь без препятствий. Найденный путь в разных местах может быть горизонтальным, вертикальным или диагональным. Если путь найден, то функция вернет значение True, а иначе - False. Точки найденного пути хранятся в списке PathList, имеющего тип TPoint. Вы можете создать свою процедуру, например: 
procedure GetPathPoints(Row, Col);
begin
 // Обработка найденной точки
end;
Если имя этой процедуры указать в качестве 5-го параметра функции Find(), то для каждого найденного элемента будет вызываться процедура GetPathPoints(), в которую будут переданы координаты найденной точки.

После поиска пути вы можете узнать время, затраченное на поиск - для этого случит переменная SearchTime.

Примеры

Выполнение поиска
  if FP.Find(5, 5, 20, 20) then ShowMessage('Путь найден');
