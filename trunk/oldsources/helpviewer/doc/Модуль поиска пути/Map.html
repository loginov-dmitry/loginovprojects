TFindPath

Map

Матрица карты

Объявление функции

property  Map[Row, Col: Integer]: Boolean read GetMapEl write SetMapEl;

Назначение

Свойство позволяет осуществлять запись и чтение элементов матрицы карты. Вы можете использовать данное свойство как обычный массив Delphi. Для определения числа строк и столбцов матрицы карты используйте соответственно свойства MapRows и MapCols.

Примеры

Запись элемента в матрицу карты
  FP.Map[1, 1] := True;
Чтение элемента матрицы карты
  B := FP.Map[1, 1];
