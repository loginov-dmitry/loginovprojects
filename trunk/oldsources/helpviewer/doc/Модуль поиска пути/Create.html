TFindPath

Create

Создание объекта

Объявление функции

  constructor TFindPath.Create(MapRows, MapCols: Integer); overload;

Назначение

Конструктор создает объект типа TFindPath. При этом создается матрица карты Map, и все необходимые переменные.

Примеры

Создание объекта
  FP := TFindPath.Create(100, 50);
