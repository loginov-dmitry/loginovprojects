Просмотр разделов справочной системы

Для просмотра разделов справочной системы предназначено приложение HelpViewer.exe, которое находится в директории Help системы Matrix. Типичное окно программы представлено на рисунке.



В левой части окна вы видите разделы справки, в правой части находится окно просмотра веб-документов, а вверху размещен набор кнопок. Приложение позволяет просматривать справки очень простого формата: справка разбивается на тематические разделы (они обозначаются иконкой закрытой или открытой книги), к которым относится множество страниц справки, например в тематический раздел "Система Matrix" включены страницы "Введение", "Структура системы", "Структура ядра" и т.д. Тематические разделы называются также, как и папки справки на диске, т.е. существуют папки с названиями "Система Matrix", "Справочная система" и т.д. Страницы справки - это файлы с расширением HTML. Если в "тематическую" папку поместить файл Describe.htm, то программа будет показывать его содержимое при щелчке на названии раздела. Если вместе с "тематическими" папками поместить файл Sort.txt, то программа будет располагать тематические разделы в соответствии с перечнем директорий, записанных в этот файл. Таким образом в файл Sort.txt записываются названия папок, причем в каждой строке можно указать только одну папку. Если вместе со страницами справки поместить файл Sort.txt, содержащий перечнень файлов, то программа отсортирует страницы справки так, как указано в этом файле. Имена файлов нужно записывать полностью с расширением, причем в каждой строке может быть указан только один файл.

Для справочной системы должен быть определен корневой каталог. По умолчанию он называется "Doc" и находится в директории с программой HelpViewer.exe. Вы можете просматривать справки, находящиеся в любом месте на жестком диске. Для этого нужно нажать кнопку "Дополнительно", перейти на вкладку "Выбор справочной системы" и указать директорию со справкой, после чего нужно нажать кнопку "Принять". Далее нужно будет обновить разделы справки путем нажатия кнопки "Обновить". С помощью данного приложения вы можете также просмотреть справочную систему MATLAB, т.к. там страницы справки тоже имеют расширение HTML.
